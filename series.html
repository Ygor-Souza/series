Crie uma votação para que os usuários escolham entre 3 séries. Cada voto atualiza o contador na tela. Também deve haver um botão para resetar a votação.

O que praticar:

  • querySelectorAll para capturar os botões "Votar"
  • addEventListener para atualizar os contadores
  • textContent para manipular os votos
  • addEventListener no botão “Resetar” para zerar todos os contadores
